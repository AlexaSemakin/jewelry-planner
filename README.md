# Jewelry Planner Telegram Bot

Telegram-бот для демонстрационного планирования производства авторских ювелирных изделий.

## Что реализовано

- создание предварительной оценки заказа через Telegram;
- индивидуальный список этапов изделия;
- справочники городов, исполнителей и маршрутов заполняются тестовыми данными при первом запуске;
- расчёт производственного плана:
  - подбор исполнителя для каждого этапа;
  - учёт города исполнителя;
  - расчёт логистики между городами;
  - учёт срока готовности;
  - вывод причины, если срок невыполним;
- запуск заказа в работу;
- список заказов;
- дашборд;
- хранение данных в PostgreSQL;
- запуск через Docker Compose.

## Быстрый старт

```bash
cp .env.example .env
```

В `.env` вставить токен от `@BotFather`:

```env
BOT_TOKEN=123456789:telegram_token
```

Запуск:

```bash
docker compose up --build
```

Остановка:

```bash
docker compose down
```

Полный сброс вместе с базой:

```bash
docker compose down -v
```

## Основной сценарий в Telegram

1. `/start`
2. `➕ Создать оценку`
3. Ввести название изделия
4. Ввести срок в днях
5. Ввести вес
6. Ввести материалы
7. Ввести этапы через запятую, например:

```text
Основа, Камни, Закрепка, Гравировка, Полировка
```

8. Бот рассчитает план, стоимость, дату готовности и объяснение.
9. Можно нажать `🚀 Запустить в работу`.

## Ограничения MVP

Это демонстрационный Telegram-прототип. Планировщик использует понятную жадную эвристику: для каждого этапа перебирает подходящих исполнителей и выбирает минимальную стоимость с учётом маршрутов и срока. Архитектура отделяет `optimizer.py`, поэтому позже можно заменить эвристику на OR-Tools/CP-SAT, не переписывая Telegram-слой.
